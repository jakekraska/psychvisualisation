Welcome

These tools will help psychologists with visualising psychological test data.

How to Use

Follow the Steps one by one.

On each tab ensure you have the correct Tests, Indexes, Subtests and Options selected prior to visualising. Each change may force the user to re-input scores.

At current, only age based SEM values from the US are available. Australasian publishers of the WISC-V, WAIS-IV and WJ IV have been contacted. Once available, an option will be added to allow plotting of confidence intervals based on US or Australian norms.

If you plot a score and it does not provide a confidence interval, it means that this score does not have SEM values for the necessary age range
